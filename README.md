### SystemAnalysis

## Installation

# Server
- git init
- git clone https://github.com/CHIMDAVIVEC/SystemAnalysis
- cd SystemAnalysis
- npm i
- move folder mongoose-auto-increment into node_modules folder with replace

# Client
- cd client
- npm i
- cd ..

# Start
- Mainly, you can run both server and client using next command:
- npm run dev
